to run my program in terminal enter:
    1. gcc --std=gnu99 -o smallsh smallsh.c          --> gcc --std=gnu99 -o <exec> <filename>
    2. chmod +x ./p3testscript                       -- > test script
    3. ./p3testscript 2>&1                           --> run test script


to run my program without the test script:
    1. gcc --std=gnu99 -o smallsh smallsh.c
    2. ./smallsh
